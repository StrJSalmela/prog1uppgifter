# Programmering Nivå 1 i Python

Detta repository innehåller olika programmeringsuppgifter gjorda under kursen i Python.

## Innehåll

Uppgifterna är baserade på område 3 till område 5 i kursen. 

### Område 3
  Område 3 fokuserar på:
  - Variabler
  - datatyper
  - input/output
  - for och while loopar
  - if villkor

### Område 4
  Område 4 fokuserar på:
  - Felsökning
  - feltyper
  - Kodkvalité
  - Kontrollstrukturer
    
### Område 5
  - Samlingsdatatyper
  - Användarinteraktion
  - Användargränssnitt
